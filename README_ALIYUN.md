# 阿里云函数计算部署 - 3分钟搞定

## 📖 超详细图文教程

### 🎯 准备工作

你需要上传的文件（在项目目录中）：
1. `index.py` （函数入口，我刚创建的）
2. `get_ai_news.py` （主程序）
3. `requirements.txt` （依赖）

## 🚀 开始部署（跟着做就行）

### 第1步：打开控制台

1. 访问：**https://fc.console.aliyun.com/**

2. 左上角选择地域：**华东1（杭州）** 或离你最近的

---

### 第2步：创建服务

1. 点击左侧菜单 **"服务及函数"**

2. 点击右上角 **"创建服务"** 按钮

3. 填写：
   - 服务名称：`ai-daily-news`
   - 描述：`每日AI新闻推送`

4. 点击 **"创建"** 按钮

✅ 服务创建完成！

---

### 第3步：创建函数

1. 在刚创建的服务中，点击 **"创建函数"**

2. 选择 **"使用内置运行时创建"**

3. 填写基本信息：
   ```
   函数名称: daily-news-push
   运行环境: Python 3.10
   请求处理程序: index.handler
   函数代码: 在线编辑
   内存规格: 256 MB
   超时时间: 300 秒
   ```

4. 点击 **"下一步"**

---

### 第4步：配置代码（重要！）

1. **在代码编辑器中，删除所有默认代码**

2. **复制以下代码并粘贴**：

```python
import sys
import os

def handler(event, context):
    """阿里云函数计算入口"""
    from get_ai_news import main

    os.environ['WECOM_WEBHOOK'] = 'https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=YOUR_WEBHOOK_KEY_HERE'
    os.environ['ENABLE_TRANSLATION'] = 'true'

    try:
        main()
        return {'statusCode': 200, 'body': 'Success'}
    except Exception as e:
        print(f"Error: {e}")
        return {'statusCode': 500, 'body': str(e)}
```

3. **点击"创建"按钮**

---

### 第5步：上传依赖文件

1. 在函数详情页，找到 **"代码"** 标签页

2. 点击 **"上传文件"** 或 **"文件上传"** 按钮

3. 依次上传：
   - `get_ai_news.py`
   - `requirements.txt`

**提示**：这两个文件在你的项目目录：`/home/yuan/Soc/V853/tina-v853-open/ai-daily-news/`

---

### 第6步：配置环境变量（重要！）

1. 在函数详情页，点击 **"配置"** 标签

2. 点击左侧 **"环境变量"**

3. 点击 **"编辑"** 或 **"添加变量"**

4. 添加以下变量：

| 变量名 | 值 |
|--------|-----|
| WECOM_WEBHOOK | https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=YOUR_WEBHOOK_KEY_HERE |
| ENABLE_TRANSLATION | true |

5. 点击 **"保存"**

---

### 第7步：创建定时触发器（重要！）

1. 在函数详情页，点击 **"触发器管理"** 标签

2. 点击 **"创建触发器"**

3. **创建第一个触发器（早上9:20）**：
   - 触发器名称：`morning-push`
   - 触发方式：**定时触发**
   - Cron表达式：`0 20 1 * * *`
   - 描述：`北京时间9:20推送`
   - 点击 **"确定"**

4. **创建第二个触发器（中午12:20）**：
   - 触发器名称：`noon-push`
   - Cron表达式：`0 20 4 * * *`
   - 描述：`北京时间12:20推送`
   - 点击 **"确定"**

5. **创建第三个触发器（下午15:50）**：
   - 触发器名称：`afternoon-push`
   - Cron表达式：`0 50 7 * * *`
   - 描述：`北京时间15:50推送`
   - 点击 **"确定"**

---

### 第8步：测试运行

1. 在函数详情页，点击右上角 **"测试函数"** 按钮

2. 点击 **"测试"**

3. 等待执行完成（约30-60秒）

4. **查看日志**：
   - 点击 **"日志"** 标签
   - 查看执行结果

5. **检查企业微信群**是否收到消息

## 🔍 常见问题

### Q: 如何查看执行日志？
A: 函数详情页 → 日志 → 选择时间 → 查看日志

### Q: 如何修改推送时间？
A: 触发器管理 → 找到触发器 → 点击编辑 → 修改Cron表达式

### Q: 推送失败怎么办？
A:
1. 查看日志找到错误信息
2. 检查环境变量是否正确配置
3. 检查企业微信webhook是否有效

### Q: 需要配置邮箱吗？
A: 暂时不需要，企业微信推送就够了
